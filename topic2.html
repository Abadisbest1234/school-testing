What is Competitive Programming?

Competitive programming is a sport where programmers solve algorithmic problems under time constraints. It develops problem-solving skills, helps you learn new algorithms and complex problems, There are many compatitons for it such as the AIPO (all ireland programming olimpiyad) which I have competed in.

Essential Algorithms

  • Sorting: QuickSort, MergeSort, HeapSort
  • Searching: Binary Search, Linear Search
  • Graph Algorithms: BFS, DFS, Dijkstra's Floyd-Warshall algorithim and grovers quantum search algorithm
  • Dynamic Programming: Memoizatio and, Tabulation
  • Mathematical: Modular Arithmetic, Prime Numbers and finding the greatest common demonanator
  • Data Structures: Arrays, Linked Lists, Trees and Hash Tables
