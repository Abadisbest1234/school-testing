Projects

8-bit Register

A digital storage device built using a 74HC595 shift register IC. Each LED represents a single bit (0 or 1), allowing binary numbers to be stored and displayed.

How it Works:

  • Data is sent serially (one bit at a time) into the 74HC595
  • The IC shifts each bit into internal storage
  • Once loaded, the latch pin updates all outputs at once
  • Each output controls an LED representing a binary value
  • This allows storage of 8-bit numbers (0–255)

Wikipedia Summarizer

A smart device that connects to Wi-Fi and fetches summarized Wikipedia content using an ESP32 microcontroller.

How it Works:

  • The ESP32 connects to a Wi-Fi network
  • User inputs a topic using a joystick interface
  • The ESP32 sends a request to Wikipedia's API
  • Text is processed and shortened into a summary
  • The summary is displayed on a TFT screen
